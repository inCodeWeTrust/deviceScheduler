approximationMode behavior

SKIP_APPROXIMATION_FAST = 0x40 ...
SKIP_APPROXIMATION_PRECISE = 0x80 ...
SKIP_APPROXIMATION_VERY_PRECISE = 0xC0 stop not until a huge amount of values are within a small tolerance
