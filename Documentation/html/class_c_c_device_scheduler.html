Detailed Description

The scheduler holds all devices and controls them by calling their startTask(), operateTask(), stopTask() etc. routines.

Function adds a control button to the control button array and returns the index of the button. A control button is a input device, that can provide either a HIGH or a LOW level at a input pin or simply connect the pin and GND using the internal inputPullup-function. Specific parameters are passed.

Parameters
buttonNamethe human-readable name of the device (used for verbose output).
button_pinthe pin number of the button's pin.
buttonActivthe state of the pin, where the button should trigger actions.
pullupif a nonzero value is passed, the input pullup is activated.
