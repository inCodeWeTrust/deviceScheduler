void stopDynamicallyBySensor ( unsigned char  sensor,
unsigned int  initiatePerformanceValue,
unsigned int  targetValue,
float  stopPerformance,
approximationMode  approximation 
)

Function sets up the task to be decelerated and stopped when a specified sensor reaches specified values. The device is decelerated when the sensor value "initiatePerformanceValue" is exceeded. The velocity while the approximation is controlled by the error amplifying value "stopPerformance" and the device is stopped when the sensor values have stabilized at "targetValue".

Parameters
sensorholds the pin number of the sensor, that should determine task process.
initiatePerformanceValuespecifies a value of the sensor from withch on the approximation process is started.
targetValuespecifies the desired target value of the sensor "sensor" where teh task is to be stopped.
stopPerformancespecifies an error amplifying factor that determines the velocity of the device when approximating the target sensor value "targetValue".
approximationspecifies the precision of the approximation. Possible values are: "SKIP_APPROXIMATION_IMMEDIATELY", "SKIP_APPROXIMATION_VERY_FAST", "SKIP_APPROXIMATION_FAST", "SKIP_APPROXIMATION_PRECISE", "SKIP_APPROXIMATION_VERY_PRECISE" and "SKIP_APPROXIMATION_NEVER"
